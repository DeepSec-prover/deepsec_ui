executable: interface_deepsec

main : interface_deepsec [options] file_name_1 ... file_name_n

  Execute the speficiations files file_name_1 ... file_name_n

Output :
  For each file, compute the hash for the name of the folder / result file and output it on the
  standard input + write into the files.

  "intialisation: name_1 .... name_n"
    name_1 ... name_n represent the hash of all the input files.

  "complete_query: name id"
    name is the hash of the file, id is the number of the query that was completed.

  "complete_run: name"

  "exit"

Input :


command :
  --version : display the version
  -v

  --show_deepsec_dir : display the path of deepsec directory

  --show_output_dir : show the path of deepsec results database

  --interactive

options :

  --deepsec_dir : setup the path of deepsec directory
  -d

  --output_dir : setup the path of deepsec results database

  --distributed <int> : Activated distributed computation
      <int> is the number of job. Cannot be zero.

  --distant_worker <string> <string> <int> : Add a distant server with <host> <path> <nb_process>

  --jobs <int>: Set up the number of jobs that we will precompute for distributed computation

  --round_timer <int>: Set up the number of second for a round.

  --default_semantics classic | private | eavesdrop
